# Aplicación movil Ecuaciclismo

<img width="1920" height="1080" alt="Ecuaciclismo Presentación final" src="https://github.com/user-attachments/assets/db2be4c3-67cc-4515-a702-c9d23b0f000e" />

Ante los altos índices de inseguridad en Ecuador —robos, asesinatos y secuestros—, las actividades deportivas al aire libre, especialmente el ciclismo, han disminuido notablemente. Como respuesta, han surgido comunidades ciclistas que buscan proteger a sus miembros rodando en grupos grandes. Sin embargo, la organización de estas salidas masivas trae nuevos desafíos que las apps de comunicación actuales no logran cubrir, como la sobrecarga de información y la falta de una estructura clara para coordinar.

En respuesta a esto, se ha desarrollado la aplicación móvil EcuaciclismoApp, en conjunto con una actividad ciclista local en Guayaquil, Ecuador. Esta aplicación tiene como principal objetivo soportar, gestionar y potenciar las actividades ciclistas de forma comunitaria, para ello los siguientes módulos han sido desarrollados:

## Tech Stack
El frontend de esta aplicación fue desarrollada con el framework React Native, para permitir su uso e instalación en dispositivos Android e iOS. Adicionalmente, el backend está construido en Django.
